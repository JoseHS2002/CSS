Introducción a CSS

  1. ¿Qué es CSS y para que sirve?

    • CSS es un lenguaje que maneja el diseño y presentación de las páginas web.

    • CSS es un lenguaje que trabaja junto con html y permite hacer muchas modificaciones en una pagina html,

      dichas modificaciones pueden estar en una página aparte para poder dejar el html más limpio.
